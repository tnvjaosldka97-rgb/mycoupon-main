# Service Worker 장애 복구 가이드

마이쿠폰 앱에서 로딩 문제, 화면 멈춤, 오래된 콘텐츠 표시 등의 문제가 발생할 경우 아래 절차를 따라 Service Worker를 수동으로 초기화할 수 있습니다.

---

## 🖥️ 데스크톱 브라우저 (Chrome, Edge, Brave 등)

### 1단계: DevTools 열기
- **Windows/Linux**: `F12` 또는 `Ctrl + Shift + I`
- **Mac**: `Cmd + Option + I`

### 2단계: Application 탭 이동
1. DevTools 상단 메뉴에서 **Application** 탭 클릭
2. 왼쪽 사이드바에서 **Service Workers** 선택

### 3단계: Service Worker 등록 해제
1. 현재 등록된 Service Worker 목록에서 마이쿠폰 도메인 찾기
2. **Unregister** 버튼 클릭

### 4단계: 캐시 삭제
1. 왼쪽 사이드바에서 **Storage** → **Cache Storage** 선택
2. 모든 캐시 항목 우클릭 → **Delete** 선택
3. 또는 상단의 **Clear site data** 버튼 클릭

### 5단계: 페이지 새로고침
- **Hard Refresh**: `Ctrl + Shift + R` (Windows/Linux) 또는 `Cmd + Shift + R` (Mac)
- 또는 주소창에서 새로고침 버튼을 길게 눌러 **"캐시 비우기 및 강력 새로고침"** 선택

---

## 📱 모바일 브라우저

### Android Chrome

#### 방법 1: 사이트 데이터 삭제
1. Chrome 주소창 왼쪽의 **자물쇠 아이콘** 또는 **정보 아이콘** 탭
2. **사이트 설정** 선택
3. **데이터 삭제** 또는 **저장소 지우기** 탭
4. 확인 후 페이지 새로고침

#### 방법 2: 브라우저 캐시 전체 삭제
1. Chrome 메뉴(⋮) → **설정**
2. **개인정보 보호 및 보안** → **인터넷 사용 기록 삭제**
3. **캐시된 이미지 및 파일** 체크
4. **데이터 삭제** 탭

### iOS Safari

#### 방법 1: 사이트별 데이터 삭제
1. 주소창 왼쪽의 **aA** 버튼 탭
2. **웹사이트 설정** 선택
3. **모든 쿠키 차단** 또는 **웹사이트 데이터 지우기** 탭
4. 페이지 새로고침

#### 방법 2: Safari 전체 캐시 삭제
1. iOS **설정** 앱 열기
2. **Safari** 선택
3. **방문 기록 및 웹사이트 데이터 지우기** 탭
4. 확인 후 Safari 재시작

### iOS Chrome

1. Chrome 메뉴(⋯) → **설정**
2. **개인정보 보호** → **인터넷 사용 기록 삭제**
3. **캐시된 이미지 및 파일** 체크
4. **인터넷 사용 기록 삭제** 탭

---

## 🔍 문제가 계속될 경우

위 방법으로 해결되지 않으면 다음을 시도하세요:

1. **브라우저 완전 종료 후 재시작**
   - 백그라운드 탭까지 모두 닫기
   - 모바일: 앱 스와이프하여 완전 종료

2. **시크릿/프라이빗 모드에서 접속**
   - 캐시/쿠키 없이 새로 시작
   - 정상 작동 시 일반 모드에서 1~5단계 재시도

3. **다른 브라우저에서 접속**
   - Chrome ↔ Safari ↔ Firefox 등 교차 테스트

4. **네트워크 연결 확인**
   - Wi-Fi ↔ 모바일 데이터 전환
   - 라우터 재시작

5. **앱 재설치 (PWA 설치 시)**
   - 홈 화면에서 앱 삭제
   - 브라우저에서 다시 "홈 화면에 추가"
